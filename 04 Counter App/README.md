# Program 4: Create a counter app with increment and decrement buttons using useState. Implement a useEffect hook to display a message when the counter updates. Add a feature to automatically reset the counter to zero after 10 seconds.

A feature-rich counter application built with React that demonstrates the use of `useState` and `useEffect` hooks. The app includes increment/decrement functionality, real-time update messages, and an automatic reset feature.

## ğŸ¨ Key Concepts Demonstrated

### 1. **useState Hook**
- Manages multiple state variables
- Handles state updates efficiently

### 2. **useEffect Hook**
- Performs side effects (timers, messages)
- Implements cleanup functions
- Responds to state changes with dependencies array

### 3. **Event Handling**
- Button click events
- State updates on user interactions

### 4. **Conditional Rendering**
- Shows/hides message based on state
- Displays timer only when counter is active

### 5. **Timer Management**
- `setTimeout` for temporary messages
- `setInterval` for countdown timer
- Proper cleanup to prevent memory leaks
