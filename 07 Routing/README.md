# Program 7: React Router Demonstration

A comprehensive React application demonstrating routing capabilities using React Router v6. This app showcases various routing concepts including basic navigation, dynamic routes, programmatic navigation, and 404 handling.

---

## âœ¨ Features

- **Basic Routing**: Navigate between different pages using React Router
- **Declarative Navigation**: Use `Link` components for client-side navigation
- **Route Configuration**: Set up routes using `Routes` and `Route` components
- **Clean UI**: Simple and intuitive user interface
- **SPA (Single Page Application)**: Fast navigation without page reloads
- **React Router v6**: Uses the latest React Router syntax and features
