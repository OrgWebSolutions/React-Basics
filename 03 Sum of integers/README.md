## ğŸ’» How It Works

### Component Structure

The application uses React's `useState` hook to manage the state of:
- `num1`: First integer input
- `num2`: Second integer input
- `sum`: Calculated result

### Key Functions

1. **calculateSum()**: 
   - Parses input values to integers
   - Handles empty inputs by treating them as 0
   - Updates the sum state

2. **handleReset()**: 
   - Clears all input fields
   - Resets the sum to null

## ğŸ¨ Features Breakdown

### State Management
- Uses React Hooks (`useState`) for managing component state
- Controlled components for input fields

### Event Handling
- `onChange` events for updating input values
- `onClick` events for button actions

### Conditional Rendering
- Result section only displays when sum is calculated
- Uses `{sum !== null && ...}` for conditional display

### Styling
- Gradient backgrounds
- Smooth animations and transitions
- Responsive design with media queries
- Modern card-based layout
